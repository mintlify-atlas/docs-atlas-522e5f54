## What is Codeplain?

Codeplain is an AI-powered code generation platform that turns **\*\*\*plain** specifications — written in plain language — into production-ready software. Describe what you want your software to do, and Codeplain handles the implementation.

The `codeplain` CLI connects to the Codeplain API, reads your `.plain` specification file, and generates fully functional, tested code — one requirement at a time.

## How it works

<Steps>
  <Step title="Write a .plain specification">
    Describe your application's requirements in ***plain — a structured plain-language format with definitions, functional specs, and acceptance tests.
  </Step>
  <Step title="Run the codeplain CLI">
    Execute `codeplain myapp.plain` and watch the interactive TUI as Codeplain generates each functional requirement.
  </Step>
  <Step title="Get production-ready code">
    Code is generated into the `plain_modules/` build folder, along with unit tests and conformance tests validated against your acceptance criteria.
  </Step>
</Steps>

## Example: Hello World in ***plain

```plain hello_world_python.plain
---
description: 'A simple Hello World Python application'
import:
  - python-console-app-template
---

***functional specs***
- :App: should print "Hello, World!" to the console
```

Run it with:

```bash
codeplain hello_world_python.plain
```

The generated code appears in `plain_modules/hello_world_python/` and the final build in `build/`.

  You need a Codeplain API key to generate code. Contact [support@codeplain.ai](mailto:support@codeplain.ai) to get yours.
</Note>
